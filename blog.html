Question-1: What are the different ways to select an element in the DOM?

We can select a element in multiple way :

  • document.getElementById("myId");
  • document.getElementsByClassName("myClass");
  • document.getElementsByTagName("div");
  • document.getElementsByName("username");
  • document.querySelector(".myClass");

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML: Returns HTML markup inside the element.
innerText: Returns visible text only
textContent: Returns all text inside the element

Question-3: What is event delegation in the DOM?

Event delegation is a technique in JavaScript where we attach a single event listener to a parent element instead of attaching separate listeners to multiple child elements.
The event “bubbles up” from the child element to the parent, and the parent can handle the event for all its children.

Why use event delegation?
Reduces memory usage by adding fewer event listeners.
Works for dynamic elements added after the page loads.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts from the innermost element (the target) and bubbles up to its parent elements in the DOM tree.
This means if you click a child element, the same event can also trigger on its parent elements unless you stop it.

Question-5: How do you create, add, and remove elements using JavaScript?

1. Create an element
document.createElement() to create a new element:
const newDiv = document.createElement("div");
newDiv.textContent = "Hello World!";

2. Add (append) the element to the DOM
appendChild() or append() is used to insert element into a parent element:
const parent = document.getElementById("container");
parent.appendChild(newDiv); // adds newDiv as the last child

Or using append() (can append multiple nodes or text): parent.append(newDiv, "Extra Text");

3. Remove an element
removeChild() or remove() to delete an element:
// Using removeChild
parent.removeChild(newDiv);
// Using remove (modern)
newDiv.remove();
